# Anti-Theft GPS Tracker

## Overview
This project is a low-cost anti-theft GPS tracking device built using an Arduino Nano and external modules.
It helps users locate stolen or lost items using real-time or logged GPS location data.

## Why this project?
Many people cannot afford commercial GPS trackers. This project focuses on:
- affordability
- privacy
- low power consumption
- educational value

## Features (Planned)
- GPS location tracking
- Movement detection
- LED status indicators
- Low power sleep mode
- Location history logging

## Hardware Used
- Arduino Nano
- GPS module (planned: NEO-6M)
- LEDs + resistors
- Breadboard
- Battery system (planned)

## Software
- Arduino C/C++
- Serial communication
- GPS data parsing

## Current Status
- Repository created
- Breadboard prototyping started
- GPS integration in progress
